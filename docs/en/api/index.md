# API Reference

This section documents the APIs available in Trix Admin.

## Store

Pinia stores for state management:

- [App Store](/en/api/store#app-store) - Application state
- [Auth Store](/en/api/store#auth-store) - Authentication state
- [Route Store](/en/api/store#route-store) - Route state
- [Theme Store](/en/api/store#theme-store) - Theme state

## Hooks

Composable functions:

- [useTable](/en/api/hooks#usetable) - Table data management
- [useForm](/en/api/hooks#useform) - Form handling
- [useLoading](/en/api/hooks#useloading) - Loading state
- [useBoolean](/en/api/hooks#useboolean) - Boolean state

## Utilities

Helper functions:

- [Storage](/en/api/utils#storage) - Local storage utilities
- [Request](/en/api/utils#request) - HTTP request utilities
- [Format](/en/api/utils#format) - Data formatting
